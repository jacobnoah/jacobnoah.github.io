Earthquake Data Project

In My Application Development class at Indiana University, I got put in a 3 team group tasked with a very difficult project. Our project was to use data from a csv file which has every Earthquake recorded in the world for a single month, and each earthquake had 27 different parameters, and we had to build a GUI to search through and filter these earthquakes by the parameters as well as show a google maps fx GUI that uses Google Maps API to be able to pin every earthquake from the requested data. I asked to be the lead programmer and challenged myself with some help from my team members and we were able to finish this project with a beautiful GUI and everything was working great.
