SportsRadar NFL API Application

During the semester of Fall 2019 I was able to take a Cordova class and learned to make apps using IOS and Android. This app was made in IOS and uses the SportsRadar NFL API to give specific play-by-play information. Using some logic I was able to make this dynamic for any week of the 2019 season. This allows the user to select and game from any week and see every play in order from start to finish.
